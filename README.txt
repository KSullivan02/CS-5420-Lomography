----------------Project 5 - Lomography------------------
This project allows the user to apply effects to their images using 2 scaling bars at the top of their image.
This involves a vignette center circle effect as well as a red channel effect controlled through a LUT

-----Usage-----
./lomo [image]

After loading the image the user can adjust the two effects using the trackbars at the top of the image.
Afterwards the user can press 's' to save the image or 'q' to quit
